- **DocumentWindowDockedEventArgs**
  - This class provides data for the DocumentWindowDocked event of the DocumentWindowCollection class.
- **DocumentWindowDockedEventHandler**
  - A delegate implemented by observers that are interested in document window docked events.

- **DocumentWindowFloatedEventHandler**

  - This is the base class for the DocumentWindowDockedEventArgs,DocumentWindowFloatedEventArgs, and DocumentWindowToBeDockedEventArgs classes.
- **DocumentWindowToBeDockedEventArgs**

- **DocumentWindowToBeDockedEventHandler**
