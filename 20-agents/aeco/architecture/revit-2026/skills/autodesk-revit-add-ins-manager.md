# Autodesk.RevitAddInsManager

- **AddInItemSettings**
  - This class represents the settings of an Add-in registered with Revit.
- **AddInsManagerSettings**
  - This class contains the settings to enable/disable loading of Add-ins registered with Revit.             Not all Add-ins can be disabled, some Add-ins installed with Revit will always be loaded.
