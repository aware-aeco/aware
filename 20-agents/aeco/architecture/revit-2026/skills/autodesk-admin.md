# Autodesk.Admin

- **AddInItemAdminLoadOptions**
  - Administrator options for Add-in item load control.
- **AddInItemAdminSettings**
  - This class represents the settings of Revit Add-ins item which is controlled by the Administrator.
- **AddInManifestAdminSettings**
  - This class represents the settings of a single Revit Add-ins manifest which is controlled by the Administrator.
- **AddInsManagerAdminSettings**
  - This class represents the settings of Revit Add-ins manager which can be controlled by the Administrator.
