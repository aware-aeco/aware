# revit-link-instance-move-base-point-to-host-base-point

Lifecycle: single

Moves this link instance so that the base point in    the linked document is aligned to the base point in the    host document. This is a one-time movement and does not    set up any shared coordinates relationship.  If the rotation angle of this link instance was changed after insertion,    the rotation angle can be preserved or reset to the original insertion angle.
