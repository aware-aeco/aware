# Layout::TableRow API reference

A TableColumn is a single row from a table.

## Methods

- `bottom_edge_style` — The #bottom_edge_style method returns the Style of a Layout::TableRow's bottom edge.
- `bottom_edge_style=` — The #bottom_edge_style= method sets the Style of a Layout::TableRow's bottom edge.
- `height` — The #height method returns the height of the Layout::TableRow.
- `height=` — The #height= method sets the height of the Layout::TableRow.
- `top_edge_style` — The #top_edge_style method returns the Style of a Layout::TableRow's top edge.
- `top_edge_style=` — The #top_edge_style= method sets the Style of a Layout::TableRow's top edge.
