# Sketchup::RenderingOptionsObserver   Abstract API reference

To implement this observer, create a Ruby class of this type, implement the desired methods, and add an instance of the observer to the objects of interests.

## Methods

- `onRenderingOptionsChanged` — The onRenderingOptionsChanged method is invoked whenever the user changes their rendering options.
