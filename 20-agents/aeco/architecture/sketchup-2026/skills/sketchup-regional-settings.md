# Sketchup::RegionalSettings API reference

The RegionalSettings module contains methods getting information about the user's locale settings.

## Methods

- `decimal_separator` — Returns the decimal character for the current user's locale.
- `list_separator` — Returns the list separator character for the current user's locale.
