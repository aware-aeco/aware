# Sketchup::LineStyles API reference

Provides access to the different line style objects in the model.

## Methods

- `[]` — The #[] method lets you retrieve a line style by name or index.
- `[]` — The #[] method lets you retrieve a line style by name or index.
- `each` — The #each method is used to iterate through all the line styles.
- `names` — The #names method return the support line styles as strings.
- `size` — The #size method returns the number of line styles that SketchUp supports.
- `to_a` — The #to_a method returns an array of all the line styles.
