# Autodesk.RevitAddIns

- **AddInArchitecture**
  - Defines the operating system architectures recognized by the Add-In utility.
- **AddInManifestUtility**
  - This static class provides access to read the contents of manifests from Revit .addin files.
- **Discipline**
  - Describes the conditions under which a particular external command will be visible in the Revit UI.
- **LanguageType**
  - Defines the language type of Revit Add-In.
- **ProductType**
  - An enumerated type containing the possible Revit product types.
- **RevitAddInApplication**
  - Represents a Revit external application.
- **RevitAddInCommand**
  - Represents a Revit external command.
- **RevitAddInDBApplication**
  - Represents a Revit DB external application.
- **RevitAddInItem**
  - Represents a single Revit Add-In.
- **RevitAddInManifest**
  - Represents the contents of a single .addin file.
- **RevitProduct**
  - Represents an installed instance of Revit on the local machine.
- **RevitProductUtility**
  - Provides access to installed versions of Revit on the local machine.
- **RevitVersion**
  - Defines the versions of Revit supported by the Add-In utility.
- **VisibilityMode**
  - Describes the conditions under which a particular external command will be visible in the Revit UI.
- **WarningType**
  - Defines types of warnings issued by Revit related to Add-In failures.
