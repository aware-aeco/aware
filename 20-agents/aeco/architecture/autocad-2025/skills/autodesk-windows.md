# Autodesk.Windows

- **CollectionsExtensions**

- **DpiValue**

- **ScaleFactor**
  - Represents a scale factor.
- **ScaleFactor2D**
  - Represents an x and y pair of scale factors in two-dimensional space.
- **ScaleFactorAction**
  - Specifies the action to take when applying a scale factor to a value.
- **ScaleFactorReduction**
  - Specifies the action to take when converting a scale factor to a lower-dimensional one.
- **ScaleFactorRounding**
  - Specifies mathematical rounding performed when applying a scale factor to a value or when converting a scale factor to another type.
