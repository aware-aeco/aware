# Geom::LatLong API reference

The LatLong class contains various methods for creating and manipulating latitude and longitude coordinates.

## Methods

- `initialize` — The new method creates a LatLong object.
- `latitude` — The Latitude method retrieves the latitude coordinate from a LatLong object.
- `longitude` — The Latitude method retrieves the longitude coordinate from a LatLong object.
- `to_a` — The #to_a method converts a LatLong object to an array of two values.
- `to_s` — The #to_s method converts a LatLong object to a String.
- `to_utm` — The to_utm method converts a LatLong object to a UTM object.
