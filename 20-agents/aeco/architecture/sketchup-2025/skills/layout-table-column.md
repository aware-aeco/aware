# Layout::TableColumn API reference

A TableColumn is a single column from a table.

## Methods

- `left_edge_style` — The #left_edge_style method returns the Style of a Layout::TableColumn's left edge. If this is the first column of the table, this method will return the style of the table border.
- `left_edge_style=` — The #left_edge_style= method sets the Style of a Layout::TableColumn's left edge. If this is the first column of the table, this method will set the style for the table border.
- `right_edge_style` — The #right_edge_style method returns the Style of a Layout::TableColumn's right edge. If this is the last column of the table, this method will return the style of the table border.
- `right_edge_style=` — The #right_edge_style= method sets the Style of a Layout::TableColumn's right edge. If this is the last column of the table, this method will set the style for the table border.
- `width` — The #width method returns the width of the Layout::TableColumn.
- `width=` — The #width= method sets the width of the Layout::TableColumn.
