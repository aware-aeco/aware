# Sketchup::Style API reference

The Style class contains methods for modifying information about a specific style. Styles are a collection of display settings that tell SketchUp how to draw the model.

## Methods

- `description` — The #description method gets the description for a Sketchup::Style.
- `description=` — The #description= method sets the description for a Sketchup::Style.
- `name` — The #name method gets the name for a Sketchup::Style.
- `name=` — The #name= method sets the name for a Sketchup::Style.
- `path` — The #path method gets the file path the Sketchup::Style was loaded from.
