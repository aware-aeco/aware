# Layout::Grid API reference

Class that references a Document's grid settings.

## Methods

- `clip_to_margins=` — The #clip_to_margins= method sets whether or not the grid is clipped to the page margins.
- `clip_to_margins?` — The #clip_to_margins? method returns whether or not the grid is clipped to the page margins.
- `in_front=` — The #in_front= method sets whether or not the grid is drawn on top of entities.
- `in_front?` — The #in_front? method returns whether or not the grid is drawn on top of entities.
- `major_color` — The #major_color method returns the Sketchup::Color for the major grid lines.
- `major_color=` — The #major_color= method sets the Sketchup::Color for the major grid lines.
- `major_spacing` — The #major_spacing method returns the major space size of the Layout::Grid.
- `major_spacing=` — The #major_spacing= method sets the major space size of the Layout::Grid.
- `minor_color` — The #minor_color method returns the Sketchup::Color for the minor grid lines.
- `minor_color=` — The #minor_color= method sets the Sketchup::Color for the minor grid lines.
- `minor_divisions` — The #minor_divisions method returns the number of minor divisions of the Layout::Grid.
- `minor_divisions=` — The #minor_divisions= method sets the number of minor divisions of the Layout::Grid.
- `print=` — The #print= method sets whether or not the Layout::Grid is printed.
- `print?` — The #print? method returns whether or not the Layout::Grid is printed.
- `show=` — The #show= method sets whether or not the Layout::Grid is visible.
- `show?` — The #show? method returns whether or not the Layout::Grid is visible.
- `show_major=` — The #show_major= method sets whether or not the major grid lines are visible.
- `show_major?` — The #show_major? method returns whether or not the major grid lines are visible.
- `show_minor=` — The #show_minor= method sets whether or not the minor grid lines are visible.
- `show_minor?` — The #show_minor? method returns whether or not the minor grid lines are visible.
