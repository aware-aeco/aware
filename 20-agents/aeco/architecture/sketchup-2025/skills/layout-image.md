# Layout::Image API reference

A raster image entity.

## Methods

- `initialize` — The #initialize method creates a new Layout::Image from a given image file.
- `clip_mask` — The #clip_mask method returns the clip mask of the Layout::Image, or nil if it does not have a clip mask.
- `clip_mask=` — clip_mask may be nil as of LayOut 2020.1.
