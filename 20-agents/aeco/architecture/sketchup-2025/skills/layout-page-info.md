# Layout::PageInfo API reference

This is the interface to a Document's paper space information. The paper size and margins, display resolution, and colors can all be accessed and set through this class.

## Methods

- `bottom_margin` — The #bottom_margin method returns the paper's bottom margin in inches.
- `bottom_margin=` — The #bottom_margin= method sets the paper's bottom margin in inches.
- `color` — The #color method returns the paper's color.
- `color=` — The #color= method sets the paper's color.
- `display_resolution` — The #display_resolution method returns the on screen rendering resolution quality.
- `display_resolution=` — The #display_resolution= method sets the on screen rendering resolution quality.
- `height` — The #height method returns the paper height in inches.
- `height=` — The #height= method sets the paper height in inches.
- `image_display_resolution` — The #image_display_resolution method returns the on screen image quality.
- `image_display_resolution=` — The #image_display_resolution= method sets the on screen image quality.
- `image_output_resolution` — The #image_output_resolution method returns the output image quality.
- `image_output_resolution=` — The #image_output_resolution= method sets the output image quality.
- `left_margin` — The #left_margin method returns the paper's left margin in inches.
- `left_margin=` — The #left_margin= method sets the paper's left margin in inches.
- `margin_color` — The #margin_color method returns the color of the paper's margin.
- `margin_color=` — The #margin_color= sets the color of paper's margin.
- `output_resolution` — The #output_resolution method returns the output rendering resolution quality.
- `output_resolution=` — The #output_resolution= method sets the output rendering resolution quality.
- `print_margins=` — The #print_margins= method sets whether to print the paper's margins.
- `print_margins?` — The #print_margins? method returns whether to print the paper's margins.
- `print_paper_color=` — The #print_paper_color= method sets whether or not the page color should be printed.
- `print_paper_color?` — The #print_paper_color? method returns whether or not the page color should be printed.
- `right_margin` — The #right_margin method returns the paper's right margin in inches.
- `right_margin=` — The #right_margin= sets the paper's right margin in inches.
- `show_margins=` — The #show_margins= method sets whether the paper's margins are visible.
- `show_margins?` — The #show_margins? method returns whether the paper's margins are visible.
- `top_margin` — The #top_margin method returns the paper's top margin in inches.
- `top_margin=` — The #top_margin method sets the paper's top margin in inches.
- `width` — The #width method returns the paper width in inches.
- `width=` — The #width= method sets the paper width in inches.
