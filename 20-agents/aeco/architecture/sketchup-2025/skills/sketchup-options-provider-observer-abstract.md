# Sketchup::OptionsProviderObserver   Abstract API reference

To implement this observer, create a Ruby class of this type, override the desired methods, and add an instance of the observer to the objects of interests.

## Methods

- `onOptionsProviderChanged` — The #onOptionsProviderChanged method is invoked when a property of an Sketchup::OptionsProvider changes.
