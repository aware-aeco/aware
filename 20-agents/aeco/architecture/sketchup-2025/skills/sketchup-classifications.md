# Sketchup::Classifications API reference

The Classifications class is a container/manager for all classifications in a model.

## Methods

- `[]` — The [] method is used to get a classification schema by name or index.
- `each` — The #each method is used to iterate through loaded classification schemas.
- `keys` — The keys method is used to get a list of keys in the Classifications class, which are the same as the names of the schemas.
- `length` — The #length method returns the number of loaded classification schemas.
- `load_schema` — The load_schema method is used to load a classification schema into a model.
- `size` — The #size method returns the number of loaded classification schemas.
- `unload_schema` — The unload_schema method is used to unload a classification schema that was previously loaded into a model.
