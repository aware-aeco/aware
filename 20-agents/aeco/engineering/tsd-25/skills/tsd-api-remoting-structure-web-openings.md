# TSD.API.Remoting.Structure.WebOpenings

- **ISimpleWebOpeningStiffener**
  - Represents a stiffener of a simple web opening
- **IWebOpeningStiffener**
  - Contains the common properties of web opening stiffeners
- **IWestokCircularWebOpeningStiffener**
  - Represents a stiffener of a circular Westok web opening
- **IWestokElongatedWebOpeningStiffener**
  - Represents a stiffener of an elongated Westok web opening
- **IWestokElongatedWebOpeningStiffenerZone**
  - Contains information about stiffener zone in the elongated Westok web opening
