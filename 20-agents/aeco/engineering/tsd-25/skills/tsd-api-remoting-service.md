# TSD.API.Remoting.Service

- **ApiException**
  - Represents an exception that occurred within API
- **ClientUnavailableException**
  - Represents an exception indicating that the Tekla Structural Designer client is not available
- **ModelNotPresentException**
  - Represents an exception indicating that the model with the requested ID is not currently active in Tekla Structural Designer
- **NoLongerSupportedException**
  - Represents an exception indicating that a method has been invoked which is no longer supported
- **NotYetSupportedException**
  - Represents an exception indicating that a method has been invoked which is not yet supported
- **TsdException**
  - Represents an exception that occurred within Tekla Structural Designer
