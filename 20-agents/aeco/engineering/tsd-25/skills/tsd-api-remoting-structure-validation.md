# TSD.API.Remoting.Structure.Validation

- **ErrorLevel**
  - Represents error severity
- **IErrorFamily**
  - Represents a collection of error groups
- **IErrorItem**
  - Represents an error, might contain sub-errors.
- **IValidationData**
  - Represents validation data - contains errors found during model validation, analysis, or design.
- **ValidationStatus**
  - Represents validation status
