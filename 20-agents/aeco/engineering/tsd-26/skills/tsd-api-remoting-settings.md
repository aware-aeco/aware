# TSD.API.Remoting.Settings

- **IBimSettings**
  - Represents the structural BIM import/export settings
- **IContinuousObjectExportSettings**
  - Represents settings for the export of continuous objects
- **IDesignSettings**
  - Represents the design settings for a model
- **IModelSettings**
  - Represents settings specific to a model
- **IUnitSettings**
  - Represents the unit settings
