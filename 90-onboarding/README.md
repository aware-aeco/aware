# First-hour guides

One per persona. Get from `git clone` (or `npm install -g @aware-aeco/cli`) to a working app in your discipline within an hour. No dev help needed.

| Persona | Guide | First app you'll ship |
|---|---|---|
| BIM manager | [bim-manager.md](./bim-manager.md) | Monday model audit |
| Designer | [designer.md](./designer.md) | Monday concept shots |
| Architect | [architect.md](./architect.md) | Cross-project sheet+RFI ball-in-court |
| Structural engineer | [structural-engineer.md](./structural-engineer.md) | Peer-review delta (TSD) |
| Detailer | [detailer.md](./detailer.md) | Issue drawing pack |

Each guide:
- Lists the tools you need installed
- Walks through `aware connect` for your discipline's integrations
- Points at the reference app in `30-apps/_examples/`
- Shows you how to customise it for your project
- Ends with the one command that ships value

If you get stuck, the audit doc (`99-feedback/2026-05-17-persona-audit/`) records exactly what the personas told us was hard. We've fixed most of it; the rest is in the open in `10-core/cli-roadmap.md`.
